# 🚀 **Müşteri Takip Sistemi (C# Windows Form + MongoDB)**

Bu repo, **C# Windows Form** ve **MongoDB** kullanılarak geliştirilmiş bir **Müşteri Takip Sistemi**'ni içermektedir. Proje, müşteri bilgileri üzerinde CRUD (Create, Read, Update, Delete) işlemleri sunmakla birlikte, dinamik listeleme ve arama gibi yeteneklere de sahiptir.

## ✨ **Kullanılan Teknolojiler**

- **C#**: Uygulama dili
- **MongoDB**: Veritabanı yönetim sistemi
- **Windows Forms**: Masaüstü uygulama arayüzü
- **Visual Studio**: Geliştirme ortamı

---

## 📊 **Proje Özellikleri**

Bu projede yer alan bazı özellikler:

- **Müşteri Listeleme:** Tüm müşterileri listeleme veya ID ile filtreleme.
- **Müşteri Ekleme:** Yeni müşteri ekleme işlemi.
- **Müşteri Silme:** Var olan müşterileri silme.
- **Müşteri Güncelleme:** Mevcut müşteri bilgilerinin düzenlenmesi.
- **ID'ye Göre Getirme:** Belirli bir müşteriyi ID bilgisi ile bulma.

### **Müşteri Kaydı Alanları:**
- ID
- İsim
- Soyisim
- Şehir
- Bakiye
- Alışveriş Tutarı
